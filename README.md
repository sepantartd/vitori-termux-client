# کلاینت سبک و سریع VLESS Reality برای Termux

کلاینتی سبک و سریع برای Termux که امکان اتصال به سرورهای **Xray Reality** را فراهم می‌کند.

## 🚀 ویژگی‌ها
- نصب سریع و خودکار روی Termux
- اجرای Xray با کانفیگ Reality
- ساخت کانفیگ برای کلاینت‌های مختلف
- ایجاد پروکسی SOCKS5 روی پورت **1080**
- امکان استفاده از پروکسی در **Telegram**، **v2rayNG** و سایر برنامه‌های پشتیبان SOCKS5
- ساختار ساده و مناسب برای کاربران مبتدی و حرفه‌ای

## 📦 پیش‌نیازها
- نصب Termux روی اندروید
- اتصال اینترنت
- کانفیگ VLESS Reality معتبر

## 🔧 نصب

```bash
pkg update
pkg install git curl -y
git clone https://github.com/sepantartd/vitori-termux-client.git
cd vitori-termux-client
bash install.sh
```

## ▶️ اجرا

```bash
bash start.sh
```

پس از اجرا، یک پروکسی **SOCKS5** روی آدرس **127.0.0.1:1080** ایجاد می‌شود.

---

# 📱 تنظیم پروکسی در تلگرام

1. وارد **Telegram** شوید.
2. به مسیر **Settings → Data and Storage → Proxy** بروید.
3. روی **Add Proxy** بزنید.
4. گزینه **SOCKS5** را انتخاب کنید.
5. مقادیر زیر را وارد کنید:

```text
Type: SOCKS5
Server: 127.0.0.1
Port: 1080
```

6. تنظیمات را ذخیره کرده و پروکسی را فعال کنید.

---

# 📱 استفاده از پروکسی در v2rayNG

1. برنامه **v2rayNG** را باز کنید.
2. روی علامت **+** در بالای صفحه بزنید.
3. گزینه **تایپ دستی [SOCKS]** را انتخاب کنید.
4. مقادیر زیر را وارد کنید:

```text
Address: 127.0.0.1
Port: 1080
```

5. روی **✔ (Save)** بزنید تا پروفایل ذخیره شود.
6. پروفایل ساخته‌شده را انتخاب کرده و روی دکمه **Connect** بزنید.

> **توجه:** گزینه **تایپ دستی [SOCKS]** در v2rayNG به‌صورت پیش‌فرض از پروتکل **SOCKS5** استفاده می‌کند و نیازی به انتخاب Type نیست.

> **نکته:** قبل از اتصال در v2rayNG، حتماً داخل Termux دستور زیر را اجرا کرده باشید:

```bash
bash start.sh
```

در غیر این صورت، v2rayNG نمی‌تواند به آدرس **127.0.0.1:1080** متصل شود.

---

## ⚠️ نکات مهم

- با بسته شدن Termux یا توقف اسکریپت، اتصال نیز قطع خواهد شد.
- برای اجرای مجدد تنها کافی است دستور زیر را اجرا کنید:

```bash
bash start.sh
```

- در تمامی برنامه‌هایی که از **SOCKS5** پشتیبانی می‌کنند نیز می‌توانید از مقادیر زیر استفاده کنید:

```text
Type: SOCKS5
Server: 127.0.0.1
Port: 1080
```
