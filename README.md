# کلاینت سبک و سریع VLESS Reality برای Termux

کلاینتی سبک و سریع برای Termux که امکان اتصال به سرورهای **Xray Reality** را فراهم می‌کند.

## 🚀 ویژگی‌ها
- نصب سریع و خودکار روی Termux
- اجرای Xray با کانفیگ Reality
- ساخت کانفیگ برای کلاینت‌های مختلف
- ایجاد پروکسی SOCKS5 روی پورت **1080**
- امکان استفاده از پروکسی در **Telegram**، **v2rayNG** و سایر برنامه‌های پشتیبان SOCKS5
- ساختار ساده و مناسب برای کاربران مبتدی و حرفه‌ای

## 📦 پیش‌نیازها
- نصب Termux روی اندروید
- اتصال اینترنت
- کانفیگ VLESS Reality معتبر

cd vitori-client

سپس:


bash start.sh


پس از اجرا، یک پروکسی **SOCKS5** روی آدرس **127.0.0.1:1080** ایجاد می‌شود.

---

# 📱 تنظیم پروکسی در تلگرام

1. وارد **Telegram** شوید.
2. به مسیر **Settings → Data and Storage → Proxy** بروید.
3. روی **Add Proxy** بزنید.
4. گزینه **SOCKS5** را انتخاب کنید.
5. مقادیر زیر را وارد کنید:

```text
Type: SOCKS5
Server: 127.0.0.1
Port: 1080
```

6. تنظیمات را ذخیره کرده و پروکسی را فعال کنید.

---

# 📱 استفاده از پروکسی در v2rayNG

1. برنامه **v2rayNG** را باز کنید.
2. روی علامت **+** در بالای صفحه بزنید.
3. گزینه **تایپ دستی [SOCKS]** را انتخاب کنید.
4. مقادیر زیر را وارد کنید:

```text
Address: 127.0.0.1
Port: 1080
```

5. روی **✔ (Save)** بزنید تا پروفایل ذخیره شود.
6. پروفایل ساخته‌شده را انتخاب کرده و روی دکمه **Connect** بزنید.

> **توجه:** گزینه **تایپ دستی [SOCKS]** در v2rayNG به‌صورت پیش‌فرض از پروتکل **SOCKS5** استفاده می‌کند و نیازی به انتخاب Type نیست.

> **نکته:** قبل از اتصال در v2rayNG، حتماً داخل Termux دستور زیر را اجرا کرده باشید:

```bash
bash start.sh
```

در غیر این صورت، v2rayNG نمی‌تواند به آدرس **127.0.0.1:1080** متصل شود.

## ⚠️ نکات مهم

- با بسته شدن Termux یا توقف اسکریپت، اتصال نیز قطع خواهد شد.
- برای اجرای مجدد تنها کافی است دستور زیر را اجرا کنید:

```bash
bash start.sh
```

- در تمامی برنامه‌هایی که از **SOCKS5** پشتیبانی می‌کنند نیز می‌توانید از مقادیر زیر استفاده کنید:

```text
Type: SOCKS5
Server: 127.0.0.1
Port: 1080
```



## ❓ FAQ — Frequently Asked Questions

### 1) بعد از اجرای bash start.sh تلگرام وصل نمی‌شود، مشکل چیه؟
معمولاً دو دلیل دارد:
- Termux بسته شده یا اسکریپت متوقف شده است  
- پورت 1080 توسط برنامه دیگری اشغال شده است  
راه‌حل: دوباره دستور زیر را اجرا کنید و مطمئن شوید Termux بسته نشود:
bash start.sh

---

### 2) چرا v2rayNG به 127.0.0.1:1080 وصل نمی‌شود؟
دو دلیل رایج:
- کلاینت Termux اجرا نشده  
- پروفایل SOCKS در v2rayNG درست تنظیم نشده  
حتماً قبل از اتصال، داخل Termux دستور زیر را اجرا کنید:
bash start.sh

---

### 3) آیا می‌توانم از این پروکسی در برنامه‌های دیگر استفاده کنم؟
بله، هر برنامه‌ای که از SOCKS5 پشتیبانی کند قابل استفاده است.  
تنظیمات عمومی:
Type: SOCKS5
Server: 127.0.0.1
Port: 1080

---

### 4) آیا با بسته شدن Termux اتصال قطع می‌شود؟
بله. Termux باید باز بماند تا Xray فعال باشد. اگر Termux بسته شود، پروکسی نیز قطع می‌شود.

---

### 5) چطور Xray را آپدیت کنم؟
پوشه xray/ را حذف کرده و نسخه جدید را جایگزین کنید.  
یا کل پروژه را دوباره کلون کنید:
rm -rf vitori-termux-client
git clone https://github.com/sepantartd/vitori-termux-client.git

---

### 6) چطور کانفیگ VLESS Reality جدید وارد کنم؟
فایل کانفیگ داخل مسیر زیر قرار دارد:
xray/config.json
آن را با کانفیگ جدید جایگزین کنید و سپس اجرا کنید:
bash start.sh

---

### 7) آیا می‌توانم پورت پروکسی را تغییر دهم؟
بله. پورت داخل فایل کانفیگ Xray قابل تغییر است:
xray/config.json
مقدار port را تغییر دهید و سپس اسکریپت را دوباره اجرا کنید.

---

### 8) چرا سرعت اتصال کم است؟
دلایل رایج:
- کیفیت اینترنت  
- شلوغی سرور  
- کانفیگ Reality اشتباه  
- استفاده از DNS نامناسب  
پیشنهاد:
- استفاده از DNSهای معتبر مثل Cloudflare  
- بررسی دوباره کانفیگ Reality  

---

### 9) آیا این کلاینت روی گوشی‌های قدیمی کار می‌کند؟
بله، چون بسیار سبک است و روی اکثر دستگاه‌های اندرویدی با Termux اجرا می‌شود.

---

### 10) چطور کلاینت را متوقف کنم؟
کافی است Termux را ببندید یا اسکریپت را متوقف کنید:
Ctrl + C
